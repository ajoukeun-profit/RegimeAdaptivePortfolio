## 슬라이드 3 — 관련 연구 / 참고 논문 (1분)

**Kim et al. (2019) 요약**

- HMM으로 시장 국면 레이블 생성
- EW 1/N을 primary benchmark로 사용
- 우리가 뭘 계승하고 뭘 확장했는지 1줄 정리

> **강조 포인트:** "논문 방법론을 그대로 쓴 게 아니라 Conv1D+LSTM으로 분류기를 고도화했다"

**Q. "왜 2019년 논문을 참고했나요?" 예상 질문 대응**

> "Kim et al. (2019)는 HMM 기반 시장 국면 레이블링을 가장 체계적으로 정립한 논문으로, 이후 regime detection 연구들도 이 방법론을 baseline으로 인용합니다. 오래됐다는 건 그만큼 검증됐다는 의미이기도 합니다.
>
> 저희 기여는 이 논문의 레이블링 방법론은 유지하되, 분류기를 Conv1D+LSTM으로 업그레이드한 것입니다. 즉 Kim et al.은 저희 베이스라인이고, 저희가 그걸 발전시킨 구조입니다.
>
> 실제로 Transformer도 실험했지만 488샘플에서 심각한 과적합이 발생했습니다. 최신 모델이 무조건 좋은 게 아니라 데이터 규모에 맞는 모델을 선택한 것이고, 그 근거가 실험으로 뒷받침됩니다."

**한 줄 요약:** "오래된 논문을 참고한 게 아니라, 그 논문을 개선한 것이 우리 기여입니다."

## 슬라이드 4 — 데이터 & 레이블링 (1.5분)

**HMM으로 Bear / Neutral / Bull 레이블 생성**

- 데이터: SPY + 크로스 에셋 40개 피처, 2012~2026
- HMM 504일 롤링 윈도우 → 3-state Gaussian HMM → Sharpe 기반 레이블 매핑
- Train / Valid / Test 시계열 분할 (70% / 15% / 15%)
  - Train: 2012~2022 (488샘플), Valid: 2022~2024 (105샘플), Test: 2024~2026 (105샘플)
  - **총 698샘플** (5일 단위 리밸런싱 기준)

> **강조 포인트:** 미래 데이터 누수 없이 시계열 분할한 것

## 슬라이드 5 — 모델 구조 (1.5분)

**Conv1D + LSTM: 왜 이 조합?**

- 입력: 30일 × 40피처 윈도우
- Conv1D → 단기 패턴 추출
- LSTM → 시계열 맥락 학습
- 출력: P(Bear) / P(Neutral) / P(Bull)

간단한 아키텍처 다이어그램 삽입 권장

> **강조 포인트:** "Transformer 대비 적은 데이터(총 698샘플, 학습 488샘플)에서 안정적으로 수렴"

## 슬라이드 7 — 포트폴리오 전략 설계 (1분)

**분류기 출력 → 투자 비중 변환**

```
SPY 비중 = P(Bull) + 0.5 × P(Neutral)
현금 비중 = 나머지
```

- 시장이 Bull이라고 판단할수록 주식을 많이 담고
- Bear라고 판단하면 자연스럽게 현금으로 대피
- 예측이 틀려도 확률적으로 비중을 조절하기 때문에 리스크 분산

> **강조 포인트:** "하드 ON/OFF가 아닌 확률적 비중 조절이라 급격한 변동 방지"

## 슬라이드 8 — 백테스트 결과: 상승장 (2분)

## 슬라이드 10 — 한계 & 향후 연구 (1분)

**정직하게 한계를 말하는 게 신뢰도에 유리**

- Neutral 클래스 Recall 0%: 중립 구간 레이블 자체가 불명확
- 2022처럼 주식+채권 동반 하락 환경에서 멀티에셋 정책(RMT)은 방어 효과 제한적
- 총 698샘플(학습 488개)로 Transformer 등 대용량 모델 적용 어려움 — 실제로 팀 내 Transformer 실험 결과 52% 정확도로 과적합 확인

> **강조 포인트:** "한계를 먼저 말하면 신뢰도 올라감. 우리가 이미 인지하고 있다는 것을 보여줄 것"

**향후 연구 방향 — 단계별**

| 개선 | 난이도 | 기대 효과 |
|---|---|---|
| 3-class → 2-class (Bear vs Non-Bear) | 낮음 | Neutral 0% 문제 해결 |
| 방어 자산 TLT → 단기채·MMF 교체 | 낮음 | 금리인상형 하락장 방어 |
| 거시지표(VIX·ISM 등) 레이블에 반영 | 중간 | HMM 레이블 품질 향상 |
| Bear 유형 분류 추가 (금리인상 vs 경기침체) | 중간 | 환경별 맞춤 방어 자산 선택 |
| End-to-end 학습 (Sharpe를 직접 loss로) | 높음 | 2단계 오류 전파 제거 |
| 강화학습(RL·PPO) 결합 | 높음 | 직접 수익 최적화 |

**Bear 유형 분류 예시 (슬라이드에 추가하면 임팩트 큼)**

```
금리인상 Bear (2022형) → 현금·MMF  (채권도 폭락하므로 TLT 피하기)
경기침체 Bear (2008형) → TLT·GLD   (금리 인하 기대로 채권 랠리)
```

**최신 연구와의 연결**

- 2025 PLOS One: 동일한 Conv1D+LSTM 구조로 포트폴리오 최적화 → 우리 방향이 맞음
- NRSM-MIA: HMM + 신경망 end-to-end 학습 → 우리 다음 단계의 목표
- LSTM + PPO (2024): 강화학습으로 직접 수익 최적화 → 장기 확장 방향
